\doxysubsection{Detailed Description}
Manages the application lifecycle, including initialization, updating, and cleanup. 

The \doxylink{classengine_1_1_application_handle}{Application\+Handle} class is responsible for managing the application\textquotesingle{}s scenes and external functions. It provides methods to initialize, update, and clean up the application.

\doxysubsection{Detailed Description}
Class for containing and executing Behaviors. 

The \doxylink{classengine_1_1_game_object}{Game\+Object} class manages a list of behaviors and provides methods to initialize, update, and handle events for these behaviors. It supports both single-\/threaded and multi-\/threaded execution based on the {\ttfamily C\+\_\+\+ENGINE\+\_\+\+USE\+\_\+\+THREADS} macro.

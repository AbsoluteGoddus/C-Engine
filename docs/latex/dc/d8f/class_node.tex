\doxysubsection{Detailed Description}
Represents a node that can contain sub-\/nodes and values. 

The \doxylink{class_node}{Node} class is used to create a hierarchical structure of nodes, where each node can contain sub-\/nodes and values. Each node has a name and can have a parent node.

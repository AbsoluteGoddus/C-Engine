\#define C\+\_\+\+ENGINE\+\_\+\+USE\+\_\+\+VKLOG}



Enables the Vulkan logging feature. 

If this macro is not defined, no Vulkan logging will be available. This will only work if {\ttfamily C\+\_\+\+ENGINE\+\_\+\+USE\+\_\+\+VULKAN} is defined. This will create a file called {\ttfamily vklog.\+log} in the working directory. A filter must be set for this feature to work.

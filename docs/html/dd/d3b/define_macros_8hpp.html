◆ C_ENGINE_USE_VKLOG

#define C_ENGINE_USE_VKLOG

Enables the Vulkan logging feature.

If this macro is not defined, no Vulkan logging will be available. This will only work if C_ENGINE_USE_VULKAN is defined. This will create a file called vklog.log in the working directory. A filter must be set for this feature to work.

See also
C_ENGINE_VKLOG_FILTER_ALL, C_ENGINE_VKLOG_FILTER_IMPORTANT, C_ENGINE_VKLOG_FILTER_ERROR

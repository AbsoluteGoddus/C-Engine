Detailed Description

Class for containing and executing Behaviors.

The GameObject class manages a list of behaviors and provides methods to initialize, update, and handle events for these behaviors. It supports both single-threaded and multi-threaded execution based on the C_ENGINE_USE_THREADS macro.
